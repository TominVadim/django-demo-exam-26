# Интернет-магазин обуви
Проект пробного варианта демонстрационного экзамена по специальности 09.02.07 "Информационные системы и программирование" в рамках Учебной практики.

## Установка и запуск

### 1. Скачать проект
```
git clone git@github.com:TominVadim/django-demo-exam-26.git
cd django-demo-exam-26
```
### 2. Полная установка (одна команда)
```
make all
```
### 3. Запустить сервер
```
make run
```
После этого открыть в браузере: http://127.0.0.1:8000/
